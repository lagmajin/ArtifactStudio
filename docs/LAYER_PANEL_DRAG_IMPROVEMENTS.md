# レイヤーパネル ドラッグ&ドロップ改善

## 実施内容

### 1. ドラッグ可能エリアの拡大

**問題:**
- スイッチ列（可視、ロック、ソロ、シャイ）をクリックしてもドラッグ移動ができなかった
- 名前エリアのみがドラッグ可能で、操作性が制限されていた

**改善:**
- スイッチ列でもドラッグを開始可能に
- レイヤー行の任意の位置をクリックしてドラッグ移動できるように

**変更箇所:**
```cpp
// mousePressEvent() 内
// 名前エリアまたはスイッチ列でドラッグを開始可能に
if (service) {
  service->selectLayer(layer->id());
  impl_->dragStartPos = event->pos();
  impl_->dragCandidateLayerId = layer->id();
}
```

---

### 2. 外部ファイルのドラッグ&ドロップ堅牢化

**改善前:**
```cpp
void dragEnterEvent(QDragEnterEvent* e) { e->acceptProposedAction(); }
void dragMoveEvent(QDragMoveEvent* e) { e->acceptProposedAction(); }
void dropEvent(QDropEvent* event) {
  // 簡易的な実装のみ
}
```

**改善後:**
- ファイル形式の事前検証（画像/ビデオ/オーディオのみ）
- nullptr チェックの強化
- エラーハンドリングの充実
- ビジュアルフィードバックの追加

---

## ユーザー体験の改善

### レイヤー並べ替え
✅ レイヤー行のどこをクリックしてもドラッグ開始可能  
✅ ドラッグ中に挿入位置を青い線で表示  
✅ ドロップで即座にレイヤー順序を変更  

### 外部ファイルからのレイヤー追加
✅ 対応形式：画像（PNG, JPG, etc）、ビデオ（MP4, MOV, etc）、オーディオ（WAV, MP3, etc）  
✅ 非対応ファイルはドラッグ時に拒否（カーソル変化）  
✅ 複数ファイルの一括ドラッグ対応

## テストチェックリスト

### レイヤー並べ替え
- [ ] 名前エリアをクリックしてドラッグ
- [ ] 可視スイッチ列をクリックしてドラッグ
- [ ] ロックスイッチ列をクリックしてドラッグ
- [ ] ソロスイッチ列をクリックしてドラッグ
- [ ] シャイスイッチ列をクリックしてドラッグ
- [ ] ドラッグ中に青い挿入インジケーターが表示される
- [ ] ドロップでレイヤー順序が変更される

### 外部ファイルドラッグ
- [ ] 画像ファイルをドラッグしてドロップ → ImageLayer が追加
- [ ] ビデオファイルをドラッグしてドロップ → VideoLayer が追加
- [ ] オーディオファイルをドラッグしてドロップ → AudioLayer が追加
- [ ] 複数ファイルを一括ドラッグ → 全て追加
- [ ] 非対応ファイル（.txt など）をドラッグ → 拒否される
