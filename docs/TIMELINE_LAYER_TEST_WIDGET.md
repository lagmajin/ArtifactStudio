# タイムラインレイヤーテストウィジェット

## 概要

プロジェクト内の実際のコンポジションとタイムラインレイヤーを使って、透明度、ブレンドモード、可視状態をテストするための専用ウィジェットです。

## 起動方法

**Test メニュー** → **Timeline Layer Test...** を選択

## 機能

### テスト対象

タイムライン上の実際のレイヤーを対象に以下をテスト：

1. **透明度（Opacity/Alpha）**
   - Solid Image Layer の色透明度を変更可能
   - 0.0〜1.0 で調整

2. **ブレンドモード（Blend Mode）**
   - 13 種類のブレンドモードをリアルタイムで変更可能
   - Normal, Add, Multiply, Screen, Overlay など

3. **可視状態（Visible）**
   - チェックボックスで表示/非表示を切り替え
   - 非表示レイヤーは合成から除外

### 主な機能

#### コンポジション選択
- プロジェクト内の全コンポジションから選択可能
- ドロップダウンで簡単切り替え
- 「Follow Current Composition」を有効にすると、メインウィンドウで編集中のコンポジションを自動追跡

#### レイヤーコントロール
- 各レイヤーごとに個別のコントロールパネル
- スクロール可能で多数のレイヤーにも対応
- レイヤー名とインデックスを表示

#### リアルタイムプレビュー
- 変更を即座に合成して表示
- チェッカーボード背景で透明度も確認可能
- 合成結果の統計情報を表示

#### タイムラインへの適用
- **Apply Changes to Timeline** ボタンで変更を実際のプロジェクトに反映
- Undo/Redo に対応（プロジェクト側で処理）

## 使用例

### 基本的なテスト手順

1. **コンポジションを選択**
   - ドロップダウンからテストしたいコンポジションを選択
   - または「Follow Current Composition」を有効にして自動追跡

2. **レイヤーコントロールを調整**
   - **Visible**: チェックボックスで表示/非表示を切り替え
   - **Opacity**: Solid Layer の透明度を調整
   - **Blend Mode**: ブレンドモードを変更

3. **結果を確認**
   - 右側のプレビューで即座に結果を確認
   - 統計情報でレイヤー数などを確認

4. **タイムラインに適用**（オプション）
   - **Apply Changes to Timeline** をクリック
   - 変更が実際のプロジェクトに反映される

### ブレンドモード比較

1. 複数のレイヤーを持つコンポジションを選択
2. 各レイヤーの Blend Mode を順に変更
3. プレビューで視覚的に比較
4. 最適な組み合わせを見つける

### 透明度アニメーションのテスト

1. Solid Image Layer の Opacity を調整
2. 複数のレイヤーで異なる透明度を設定
3. 合成結果を確認
4. 必要に応じてタイムラインに適用

## キーボードショートカット

| キー | 機能 |
|------|------|
| `F5` | レイヤーコントロールをリロード |

## ファイル構成

| ファイル | 説明 |
|---------|------|
| `ArtifactTimelineLayerTestWidget.ixx` | ヘッダーファイル |
| `ArtifactTimelineLayerTestWidget.cppm` | 実装ファイル |
| `ArtifactTestMenu.cppm` | メニュー統合 |

## 対応レイヤータイプ

- **Solid Image Layer**: 透明度 + ブレンドモード + 可視状態 ✅
- **Image Layer**: ブレンドモード + 可視状態 ✅
- **Video Layer**: ブレンドモード + 可視状態 ✅
- **Audio Layer**: 可視状態のみ（オーディオは合成対象外）

## 注意事項

- 実際のプロジェクトに変更を適用する場合は、必ず保存してください
- 多数のレイヤーがある場合、スクロールしてコントロールを表示します
- プレビューは 800x600 にリサイズされます
