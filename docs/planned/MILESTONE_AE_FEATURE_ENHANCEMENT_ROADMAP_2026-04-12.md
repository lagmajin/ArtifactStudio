# MILESTONE_AE_FEATURE_ENHANCEMENT_ROADMAP_2026-04-12

## After Effects 風アプリ機能向上 全体ロードマップ

本ドキュメントは ArtifactStudio を After Effects に匹敵する制作ツールへと育成するための
機能別マイルストーン一覧である。各 Phase は依存関係のある順序で配置されており、
下位 Phase を飛ばして上位に着手することは推奨されない。

---

## 現状評価 (AE 基準 2026年4月現在)

| 分類 | 実装率 | 備考 |
|------|--------|------|
| コンポジションエンジン | **85%** | トラックマット/ブレンドモード完全版が不足 |
| タイムラインUI | **78%** | グラフエディタ補間/マーカーが不足 |
| レイヤーシステム | **82%** | 調整レイヤー/親子伝搬が不足 |
| エフェクトシステム | **45%** | 標準エフェクト実装不足 |
| シェイプレイヤー | **35%** | パスアニメーション/シェイプ演算子が不足 |
| キーフレーム補間 | **20%** | イージングカーブ/モーションブラーが不足 |
| エクスプレッション | **15%** | 言語仕様/ランタイムが不足 |
| プラグインAPI | **0%** | サードパーティ互換なし |

**総合スコア:**
- 見た目の AE 度: **8/10**
- 制作体験の AE 度: **5/10**
- エンジン性能: **6/10**
- 拡張性: **8/10**

## Phase 1: 制作必須基盤 (M13 / 2026年4月〜5月)

AE として**最低限制作に使えるレベル**に引き上げる。ユーザーが初めて触ったときに
「これなら AE の代わりに使えるかも」と感じられる水準。

### P1-1: トラックマット完全版
- [ ] アルファマット / ルミナンスマット / 反転の完全スタック評価
- [ ] マット元レイの非表示プロパティ維持
- [ ] マット連鎖（AがBのマット、BがCのマット）の正しい評価順序
- **影響:** コンポジションの合成表現力が AE に近づく
- **工数:** 20-30h
- **依存:** `ArtifactCore/include/Track/`, `ArtifactCore/include/Mask/`

### P1-2: イージング付きキーフレーム補間
- [ ] ベジェカーブ補間（Temporal Interpolation）
- [ ] イーズイン / イーズアウト / イーズインアウト プリセット
- [ ] Hold キーフレーム（ステップ補間）
- [ ] グラフエディタでのベジェハンドル操作UI
- [ ] 速度グラフ表示（Temporal Graph → Speed Graph 切替）
- **影響:** アニメーションの品質が劇的に向上
- **工数:** 40-60h
- **依存:** `ArtifactCore/include/Animation/`, `Artifact/src/Widgets/ArtifactCurveEditorWidget.ixx`

### P1-3: モーションブラー
- [ ] シャッター角度設定（AE互換 0-720°）
- [ ] シャッター位相（-180°〜+180°）
- [ ] レイヤー単位 ON/OFF
- [ ] コンポジション全体のモーションブラー有効/無効
- [ ] サンプル数設定（品質/速度トレードオフ）
- **影響:** 動きのリアル感が向上
- **工数:** 25-35h
- **依存:** `ArtifactCore/include/Render/`, `ArtifactCore/include/Graphics/Effect/`

### P1-4: 調整レイヤー
- [ ] 調整レイヤー作成（下の全レイヤーにエフェクトを適用）
- [ ] 調整レイヤーのレンダリング順序（通常レイヤーの後、最終出力の前）
- [ ] 調整レイヤーのマット/トラックマット対応
- **影響:** カラーグレーディングや全体エフェクトの適用が可能に
- **工数:** 15-20h
- **依存:** `Artifact/include/Layer/ArtifactAdjustableLayer.ixx`

### P1-5: ブレンドモード完全版
- [ ] 16種類以上のブレンドモード実装（Normal, Dissolve, Multiply, Screen, Overlay 等）
- [ ] ブレンドモードのGPU実装（HLSLシェーダー）
- [ ] ブレンドモードのプレビュー更新（リアルタイム反映）
- **影響:** 合成の表現力が向上
- **工数:** 30-40h
- **依存:** `ArtifactCore/include/Layer/LayerBlend.ixx`

## Phase 2: 標準制作機能 (M14 / 2026年5月〜6月)

Phase 1 の基盤の上に、AE の**標準的な制作フロー**を再現する。

### P2-1: シェイプレイヤー演算子
- [ ] パスアニメーション（ベジェパスのキーフレーム）
- [ ] シェイプ演算子（Add, Subtract, Intersect, Exclude Overlaps）
- [ ] トリムパス（Start/End/Offset のアニメーション）
- [ ] リピーター（Transform付きの複製）
- [ ] パスに沿ったテキスト/レイヤー配置
- **影響:** モーショングラフィックスの自由度が向上
- **工数:** 50-70h
- **依存:** `ArtifactCore/include/Shape/`

### P2-2: マーカーシステム
- [ ] コンポジションマーカー（タイムライン上の注釈）
- [ ] レイヤーマーカー（レイヤー内のセグメント注釈）
- [ ] マーカーのナビゲーション（次/前マーカーへジャンプ）
- [ ] マーカーコメント表示
- **影響:** 編集作業の効率が向上
- **工数:** 15-25h
- **依存:** `ArtifactCore/include/Time/`, `Artifact/src/Widgets/Timeline/`

### P2-3: レイヤースタイル
- [ ] ドロップシャドウ
- [ ] 光彩（外側/内側）
- [ ] ベベルとエンボス
- [ ] 境界線（ストローク）
- [ ] カラーオーバーレイ
- [ ] グラデーションオーバーレイ
- [ ] パターンオーバーレイ
- **影響:** テキストやシェイプの装飾が容易に
- **工数:** 40-50h
- **依存:** `ArtifactCore/include/ImageProcessing/`

### P2-4: ワークエリア完全版
- [ ] レンダリング範囲指定（In/Out 以外のワークエリア）
- [ ] ワークエリアの時間伸縮（A/B点ドラッグで範囲変更）
- [ ] プレビュー範囲とレンダリング範囲の分離
- [ ] ワークエリアのループ/ワンショット切替
- **影響:** プレビューと本番レンダリングの使い分けが可能に
- **工数:** 20-30h
- **依存:** `ArtifactCore/include/Composition/`

### P2-5: エフェクト種類増加（GPU Parity Phase 1）
- [ ] Gaussian Blur (GPU)
- [ ] Fast Box Blur (GPU)
- [ ] Unsharp Mask (GPU)
- [ ] Curves (GPU)
- [ ] Levels (GPU)
- [ ] Hue/Saturation (GPU)
- [ ] Brightness & Contrast (GPU)
- [ ] Chroma Key (GPU)
- **影響:** 基本的なカラーコレクションが可能に
- **工数:** 60-80h
- **依存:** `ArtifactCore/include/Graphics/Effect/`, `Artifact/shaders/`

## Phase 3: 上級制作機能 (M15 / 2026年6月〜8月)

AE の**上級者向け機能**を実装。プロの制作現場で必要とされる機能群。

### P3-1: エクスプレッションエンジン
- [ ] JavaScript 互換ランタイム（Duktape 或いは V8 埋め込み）
- [ ] `time`, `value`, `linear()`, `ease()`, `wiggle()` 等のAE互換関数
- [ ] プロパティへのエクスプレッション適用
- [ ] エクスプレッションエディタ（シンタックスハイライト/補完）
- [ ] エクスプレッションのデバッグ（変数ウォッチ/エラー表示）
- **影響:** 手続き的アニメーションが可能に
- **工数:** 80-120h
- **依存:** `ArtifactCore/include/Script/`, `libs/` (JSエンジン追加の可能性)

### P3-2: 親子階層完全版
- [ ] 複数親のサポート（AEのNullオブジェクト経由階層の正式サポート）
- [ ] 親子関係のビジュアル表示（タイムライン上のインデント/接続線）
- [ ] 階層 Transform の正しい伝搬（スケールの継承設定含む）
- [ ] ピンツール（特定の点を基準にした親子関係）
- **影響:** 複雑なアニメーションリグが可能に
- **工数:** 30-40h
- **依存:** `ArtifactCore/include/Layer/`, `ArtifactCore/include/Transform/`

### P3-3: プリコンポーズ完全版
- [ ] プリコンポーズのネスト（プリコンポーズの中にプリコンポーズ）
- [ ] プリコンポーズの属性維持（元のレイヤー属性を保持）
- [ ] プリコンポーズ編集（ダブルクリックで別タブで開く）
- [ ] プリコンポーズのレンダリング順序の正しい評価
- **影響:** 大規模コンポジションの管理が可能に
- **工数:** 25-35h
- **依存:** `ArtifactCore/include/Composition/PreCompose.ixx`

### P3-4: フレームブレンディング
- [ ] フレームミックス（前後フレームのアルファブレンディング）
- [ ] ピクセルモーション（Optical Flow 相当の簡易実装）
- [ ] フレームブレンディングの ON/OFF（レイヤー単位/コンポ単位）
- [ ] スローモーション時の品質設定
- **影響:** タイムリマップ時の滑らかさが向上
- **工数:** 35-50h
- **依存:** `ArtifactCore/include/Time/TimeRemap.ixx`

### P3-5: 3D カメラトラッカー
- [ ] 2D 特徴点トラッキング
- [ ] 3D カメラソルバ（特徴点から3Dカメラ位置を復元）
- [ ] トラッキングデータへの Null/カメラレイヤー適用
- [ ] トラッキングの平滑化/外れ値除去
- **影響:** VFX 合成の精度が向上
- **工数:** 60-80h
- **依存:** `ArtifactCore/include/Tracking/`, OpenCV

## Phase 4: 拡張性・自動化 (M16 / 2026年8月〜10月)

サードパーティや自動化による**エコシステム構築**の基盤。

### P4-1: プラグイン SDK
- [ ] サードパーティエフェクトプラグインの読み込み（.aex 互換 or 独自形式）
- [ ] プラグインパラメータのUI自動生成
- [ ] プラグインのGPU/CPU バックエンド選択
- [ ] プラグインのバージョン管理/互換性チェック
- **影響:** コミュニティ/ベンダーによる機能拡張が可能に
- **工数:** 80-120h
- **依存:** `ArtifactCore/include/ImageProcessing/`

### P4-2: Python API 完全版
- [ ] pybind11 経由の全レイヤー/コンポジション/エフェクトAPI公開
- [ ] Python コンソール（REPL + スクリプト実行）
- [ ] Python スクリプトの保存/読み込み/実行履歴
- [ ] スクリプトによるバッチ処理（連番レンダリング、一括変換等）
- **影響:** 自動化パイプラインの構築が可能に
- **工数:** 40-60h
- **依存:** `Artifact/src/Script/`, `Artifact/include/Service/`

### P4-3: テンプレートシステム
- [ ] コンポジションテンプレート（.mogrt 相当）
- [ ] パラメータ公開（テンプレート利用時に編集可能なプロパティ指定）
- [ ] テンプレートブラウザ（プレビュー付き）
- [ ] テンプレートのインポート/エクスポート
- **影響:** 再利用可能なモーショングラフィックス資産の構築
- **工数:** 30-50h
- **依存:** `ArtifactCore/include/Composition/`, `ArtifactCore/include/Property/`

### P4-4: AI 機能統合
- [ ] AI アシストによる自動キーフレーム生成（動きのトラッキング→キーフレーム化）
- [ ] AI によるカラーグレーディング提案（シーン分析→LUT/パラメータ提案）
- [ ] AI によるエフェクト推奨（コンテンツに応じたエフェクト提案）
- [ ] MCP サーバー経由の外部 AI ツール連携
- **影響:** 制作の効率が大幅に向上
- **工数:** 60-100h
- **依存:** `ArtifactCore/include/AI/`, MCPブリッジ

---

## Phase 5: プロ品質 (M17+ / 2026年10月以降)

放送/映画レベルの品質を要求する**プロフェッショナル向け機能**。

### P5-1: カラーマネジメント
- [ ] OCIO (OpenColorIO) 統合
- [ ] カラースペース変換（sRGB, Rec.709, Rec.2020, ACES）
- [ ] 32bit Float HDR パイプライン
- [ ] LUT 適用（.cube, .3dl）
- **工数:** 50-70h

### P5-2: ディープコンポジット
- [ ] OpenEXR deep sample 対応
- [ ] Deep Merge / Deep Holdout
- [ ] Z-depth ベースの合成
- **工数:** 60-80h

### P5-3: チームプロジェクト
- [ ] クラウド/共有ストレージ上での複数人同時編集
- [ ] 変更のマージ/コンフリクト解決
- [ ] バージョン管理（プロジェクト履歴の分岐/マージ）
- **工数:** 100-150h

### P5-4: パフォーマンス最適化
- [ ] マルチフレームレンダリング（複数フレームの並列レンダリング）
- [ ] GPU アクセラレーション完全版（Effect 100% GPU オフロード）
- [ ] メモリ使用量の最適化（大規模コンポジション対応）
- [ ] プロキシ/低解像度プレビューの完全版
- **工数:** 80-120h
