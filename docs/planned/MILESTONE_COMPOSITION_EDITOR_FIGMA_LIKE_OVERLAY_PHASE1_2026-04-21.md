# Phase 1: Snap / Smart Guides

> 2026-04-21 作成

## Goal

コンポジットエディタのドラッグ・リサイズ・移動に、Figma 風のスナップとガイドを足す。

## Scope

- 辺合わせ
- 中心合わせ
- 同幅 / 同高
- 間隔一致
- スナップ候補の簡易表示

## Notes

- 強制吸着ではなく補助スナップとして始める
- 既存の gizmo / drag path を再利用する
- ガイド表示は描画本体に寄せる

## Success Criteria

- 操作中にガイドが出る
- スナップの有無が見える
- 既存の移動 / 拡縮操作を壊さない
