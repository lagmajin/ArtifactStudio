# マイルストーン: Illustrator AI/EPS/PDF ファイルインポート

作成日: 2026-04-18
優先度: 🟠 高
対象バージョン: M13

---

## 概要

既に完成しているベクターレンダリングシステムを活用し、Adobe Illustrator 形式 (.ai) / EPS / PDF ファイルをネイティブレイヤーとしてインポート可能にする実装計画です。

**ベクターエンジン側は100%完成しているため、この機能はファイルパーサーの追加だけで実装可能です。**

---

## 実装方式

### 推奨アプローチ: MuPDF ベース

✅ 最も現実的で実績のある方法
```
AI/EPS/PDF ファイル → MuPDF でレンダリング & パス抽出 → 内部シェイプレイヤーに変換
```

#### 長所
- ほぼ全ての AI/EPS/PDF ファイルが正しく読み込める
- テキスト、ベクターパス、イメージ、クリッピングマスクを全て抽出可能
- ライセンスは AGPL または商用ライセンス
- 既に OpenCV が入っているので依存関係の相性が良い

### 代替案
- `podofo` : PDF専用、パス抽出は得意だがAI非対応
- `libgs` (Ghostscript) : 最も互換性が高いがライセンスが厳しい
- `librsvg` : SVG専用、AIは不可
