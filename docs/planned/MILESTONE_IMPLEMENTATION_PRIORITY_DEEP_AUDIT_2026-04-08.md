**最優先。これが無いと何も始まらない**

✅ 現状:
- 線形補間しか実装されていない
- ベジェハンドルのデータ構造は既にある

✅ 必要な作業:
1.  `AnimatedValue` クラスにベジェ補間ロジックを実装
2.  イージング定数 (EasyEase / EasyIn / EasyOut) を追加
3.  速度グラフ / 値グラフ の切り替えロジック
4.  グラフエディタUIの描画

✅ 所要工数: 3日
✅ インパクト: 最大。全体の使用感が 10倍良くなる。
