# Misc Feature Ideas

このファイルは、まだ正式なマイルストーンに切り出す前の「実装するかもしれない候補」を雑に置いておくメモです。

## Layer / Timeline

- レイヤーをグループにまとめて折りたたむ
  - AE のプリコンプなしで、複数レイヤーをひとまとめに見せたい
  - 表示整理と操作単位の両方に使えるとよい
  - 関連候補: `MILESTONE_LAYER_GROUP_SYSTEM_2026-03-27.md`

- 複数レイヤーを同時にトリム
  - in/out を一括で詰める、伸ばす、揃える
  - タイムライン操作の効率化に効く

- エフェクトの on/off を一括トグル
  - デバッグ用
  - 選択レイヤー、選択エフェクト、全体の切替を想定

## Context Menu / Inspector

- 右クリックメニューのカスタマイズ
  - レイヤー種別ごとに出す項目を変えたい
  - ユーザーがよく使うコマンドを前面に出せるようにしたい

- カラーグレーディングの「強度」スライダーを全エフェクトに共通で追加
  - 各エフェクトの効き具合を統一的に制御したい
  - デバッグにも本番にも使える可能性がある

## Compare / Diff

- バージョン間の diff 表示
  - 何が変わったかを視覚的に見せたい
  - レイヤー、プロパティ、ショット、レンダー結果などを比較対象にできるとよい
  - 関連候補: `MILESTONE_REVIEW_WORKSPACE_2026-04-03.md`

## Export / Render

- 複数解像度を一括出力
  - SNS 用、クライアント用、マスターを同時に出したい
  - プリセットベースのバッチ出力があると便利

- レンダー完了を Slack / Discord に通知
  - 長時間ジョブの完了確認をしやすくする
  - ローカル完了通知とリモート通知を分けられるとよい

## Health / Diagnostics

- プロジェクトの健康診断を定期実行してレポート生成
  - 不足ファイル、壊れた参照、未使用アセット、重い構成を検出したい
  - 定期実行して履歴を残せると便利
  - 関連候補: `MILESTONE_PROJECT_FILE_VALIDATION_SPELL_CHECK_2026-03-26.md`

## Notes

- まとまったら、それぞれを正式な milestone / phase に分ける。
- 既存の正式マイルストーンと重なるものは、そちらを優先して参照する。
