## 📊 技術的特徴

### ✅ 重複排除機構

```cpp
// 保存直前にハッシュ比較
const QByteArray snapshotHash = QCryptographicHash::hash(snapshotBytes, QCryptographicHash::Sha256).toHex();

if (snapshotHash == lastCommittedSnapshotHash_) {
    // 前回と全く同一の場合はコミットしない
    return true;
}
```

✅ 全く同じ内容ならスナップショットは作成されません。
✅ 無駄なコミットを完全に排除。

### ✅ JSON ディープ比較

完全な再帰的差分抽出アルゴリズムが実装済み:
- ✅ オブジェクト階層の差分
- ✅ 配列順序の差分
- ✅ 浮動小数点値の許容誤差比較 (1e-6)
- ✅ 変更パスの記録
